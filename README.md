## ğŸ“ Support & Troubleshooting

### Common Issues

**Mobile menu not closing?**
- Check HTML data attributes
- Verify JavaScript is loaded
- Clear browser cache

**Images not responsive?**
- Ensure img-holder class is used
- Check image paths
- Verify responsive.css is loaded

**Form too small on mobile?**
- Check input font-size (must be 16px for iOS)
- Verify min-height (44px recommended)
- Test on actual device

**Layout breaks at certain width?**
- Clear CSS cache (Ctrl+Shift+Delete)
- Test in incognito mode
- Check media queries
