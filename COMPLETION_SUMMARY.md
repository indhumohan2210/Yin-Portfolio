## ğŸ” Quick Troubleshooting

### Issue: Menu not closing on mobile
**Solution**: Check console for JS errors, verify HTML data attributes

### Issue: Images not responsive
**Solution**: Use `img-holder` class, check image paths, ensure alt text

### Issue: Form too small on mobile
**Solution**: Check input font-size (16px to prevent zoom), verify min-height

### Issue: Layout broken at certain width
**Solution**: Clear cache, test in incognito, check CSS media queries
